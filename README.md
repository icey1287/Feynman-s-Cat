# 🐱 费曼的猫 (Feynman's Cat)

> 最好的学习方式是"教别人" —— 费曼学习法

## 📖 项目简介

费曼的猫是一个基于费曼学习法的AI学习应用。与传统的AI教学不同，这里是**你来教AI**，而不是AI来教你。

### 核心理念

- **痛点**：现在的AI老师都在"填鸭式"教人，用户只是被动接收，很容易走神
- **玩法**：用户学完一个知识点，必须用自己的话讲给一只"笨笨的AI电子宠物"听
- **目标**：直到用户能把这个AI彻底教会，才算通关

### 知识掌握度系统

AI宠物有一个知识条（0-100），根据数值对应三种状态：

| 知识值 | 状态 | 描述 |
|--------|------|------|
| 0-49 | 😵‍💫 一头雾水 | 对这个知识点完全不理解 |
| 50-99 | 🤔 一知半解 | 稍微熟悉了，但还是不太会用 |
| 100 | 😸 完全掌握 | 恭喜！AI被你教会了！ |

### AI的特点

- AI不是判断你对不对，而是**装傻**
- AI会根据你的描述，问一些**刁钻的"弱智问题"**来暴露你逻辑的漏洞
- 每次有效的解释都会增加知识条的进度

## 🚀 快速开始

### 1. 配置环境变量

编辑 `.env` 文件，填写你的API配置：

```bash
# OpenAI API密钥
API_KEY=your-api-key-here

# API基础URL (支持OpenAI兼容的API,以豆包为例)
BASE_URL=https://ark.cn-beijing.volces.com/api/v3

# 模型名称（以豆包为例）
MODEL=doubao-seed-1-6-flash-250828
```


### 2. 安装依赖

```bash
pip install -r requirements.txt
```

### 3. 启动服务

```bash
python backend/app.py
```

### 4. 访问应用

打开浏览器访问：http://localhost:5001

## 🎮 使用方法

1. **选择知识点**：输入你想要复习/学习的知识点（如"二叉树"、"光合作用"）
2. **开始教学**：用自己的话向AI猫咪解释这个概念
3. **回答问题**：AI会问一些"傻问题"，耐心解答
4. **提升知识条**：解释得越清楚，知识条涨得越多
5. **通关**：当知识条达到100时，恭喜你真正掌握了这个知识点！

## 💡 费曼学习法

> "如果你不能简单地解释一件事，说明你还没有真正理解它。" —— 理查德·费曼

费曼学习法的四个步骤：
1. **选择概念**：选择一个你想要学习的概念
2. **假装教学**：假装你在教一个小孩子这个概念
3. **发现漏洞**：当你卡住时，回去重新学习
4. **简化语言**：用最简单的语言重新解释
